# TTS Dialogue Scripts
## บทสนทนาสำหรับทดสอบ Multi-Speaker TTS (Gemini)

---

## 🎭 บทสนทนาสั้นๆ (1-2 นาที)

### 1. ทักทายที่ร้านกาแฟ
```
สมชาย: สวัสดีครับ รับอะไรดีครับ
มานี: สวัสดีค่ะ ขอชาไทยร้อนๆ หน่อยค่ะ
สมชาย: ได้เลยครับ อยากได้แบบไหนครับ
มานี: ชาเย็นค่ะ ไม่ต้องใส่น้ำแข็ง
สมชาย: รอแป๊บนะครับ กำลังทำให้ครับ
มานี: ไม่เร่งครับ ขอบคุณค่ะ
```

### 2. สอบถามเรื่องงาน
```
สมชาย: วันนี้เจอกันตอนไหนดีครับ
มานี: ตอนเที่ยงค่ะ เรามีเรื่องจะปรึกษา
สมชาย: เรื่องอะไรเหรอครับ น่าสนใจแล้ว
มานี: เรื่องโปรเจคใหม่ค่ะ อยากให้คุณช่วยดูให้
สมชาย: แน่นอนครับ พร้อมช่วยเสมอ
มานี: ขอบคุณมากค่ะ รู้สึกดีใจจังเลย
```

### 3. นัดเที่ยวกัน
```
สมชาย: เสาร์นี้ว่างไหมครับ
มานี: ว่างค่ะ มีอะไรจะทำกันหรือเปล่า
สมชาย: อยากชวนไปดูหนังครับ
มานี: หนังเรื่องไหนเหรอครับ
สมชาย: หนังใหม่ที่เพิ่งเข้าฉายครับ
มานี: ดีจังค่ะ ตกลงเลย
```

### 4. คุยเรื่องอาหาร
```
สมชาย: กินข้าวรึยังครับ
มานี: ยังไม่ได้กินเลยค่ะ หิวมาก
สมชาย: แล้วไปกินกันไหมครับ
มานี: ไปที่ไหนดีครับ
สมชาย: ไปที่ร้านอาหารที่ชอบไหมครับ
มานี: ดีค่ะ ไปเลย
```

## 🎪 บทสนทนาตลกขบขัน (2-3 นาที)

### 1. แมวกับปลาทูน่า
```
สมชาย: เมื่อวานนี้แมวผมทำอะไรไม่รู้
มานี: ทำอะไรล่ะครับ เล่าให้ฟังหน่อย
สมชาย: มันกระโดดเข้ามากัดปลาทูนะ
มานี: แล้วไง ปลาทูนะเป็นอะไร
สมชาย: ก็ปลาทูนะในกระป๋องน่ะสิ
มานี: แล้วมันทำอะไรต่อ
สมชาย: มันนั่งมองปลาทูนะที่ว่ายนิ่งๆ
มานี: แล้วล่ะครับ ตลกจัง
สมชาย: เหมือนคิดว่าเป็นลูกของมัน
มานี: แมวเราก็เลยเหมือนกันเลย
สมชาย: แต่เราไม่ได้กัดปลาทูนะนะ
มานี: 555+ โอ๊ย ขำมาก
```

### 2. คุณย่ากับสมาร์ทโฟน
```
สมชาย: คุณย่าเพิ่งได้สมาร์ทโฟนใหม่
มานี: โอ๊ย ดีใจจังเลย
สมชาย: แต่ไม่รู้จะใช้ยังไง
มานี: สอนให้ใช้เป็นไหม
สมชาย: พอดีเลยครับ
มานี: กดปุ่มเขียวเพื่อโทรออก
สมชาย: แล้วปุ่มสีแดงล่ะครับ
มานี: ปุ่มสีแดงเพื่อปิด
สมชาย: คุณย่าถามว่ามีคนอยู่ในกล่อง
มานี: คุณย่าคิดว่าเป็นผีหรือเปล่า
สมชาย: ต้องอธิบายยาวเลย
มานี: คุณย่าน่ารักจัง
```

### 3. หมากับไปรษณียบัตร
```
สมชาย: หมาของเพื่อนจอมซนมาก
มานี: ทำอะไรล่ะครับ
สมชาย: มันเอาไปรษณียบัตรไปกัด
มานี: แล้วไง ทำไมล่ะ
สมชาย: ไม่รู้ คงคิดว่าของกิน
มานี: แล้วเจ้าของเป็นอย่างไรบ้าง
สมชาย: ต้องไล่หมาทั้งวัน
มานี: ขำมาก หมาก็น่ารักด้วย
สมชาย: แต่ค่าไปรษณีย์แพง
มานี: ต้องเอาประกันไว้
สมชาย: ประกันหมาหรือประกันจดหมาย
มานี: 555+ คุณตลกจัง
```

## 💼 บทสนทนาทางการ (2-3 นาที)

### 1. นัดประชุม
```
สมชาย: สวัสดีครับ คุณมานี
มานี: สวัสดีค่ะ คุณสมชาย
สมชาย: มีเรื่องจะนัดประชุมพรุ่งนี้
มานี: ตอนกี่โมงครับ
สมชาย: สองโมงเช้าครับ
มานี: ที่ห้องประชุมใหญ่ใช่ไหมครับ
สมชาย: ใช่ครับ อย่าลืมเอาเอกสารไปด้วย
มานี: ได้เลยครับ จะเตรียมให้ครบถ้วน
สมชาย: ขอบคุณมากครับ
มานี: ไม่เป็นไรครับ เป็นหน้าที่
```

### 2. สั่งซื้อสินค้า
```
สมชาย: สวัสดีครับ อยากสั่งสินค้า
มานี: สวัสดีค่ะ สั่งอะไรได้ครับ
สมชาย: อยากสั่งปากกา 100 ด้าม
มานี: รุ่นไหนเป็นที่สุดครับ
สมชาย: รุ่นใหม่ที่เพิ่งออกครับ
มานี: ราคาด้ามละ 50 บาทครับ
สมชาย: รวมเป็นเท่าไหร่ครับ
มานี: รวม 5000 บาทครับ
สมชาย: ส่งเมื่อไรรับได้ครับ
มานี: 3 วันหลังจากโอนเงินครับ
สมชาย: ขอบคุณครับ
```

### 3. สัมภาษณ์งาน
```
สมชาย: สวัสดีครับ มาสัมภาษณ์ใช่ไหมครับ
มานี: สวัสดีค่ะ ใช่ค่ะ
สมชาย: ทำงานอะไรมาก่อนครับ
มานี: เคยทำงานบริการลูกค้าค่ะ
สมชาย: ทำงานมานานแค่ไหนครับ
มานี: 3 ปีค่ะ
สมชาย: ทำไมถึงอยากมาทำงานที่นี่
มานี: อยากเรียนรู้และพัฒนาตัวเองค่ะ
สมชาย: ดีมากครับ มีคำถามอะไรไหม
มานี: ไม่มีค่ะ ขอบคุณครับ
```

## 🎬 บทสนทนานำทาง (3-4 นาที)

### 1. ถามทาง
```
สมชาย: ขอโทษนะครับ ถามทางหน่อย
มานี: ไปที่ไหนได้บ้างครับ
สมชาย: อยากไปโรงพยาบาลกรุงเทพ
มานี: โรงพยาบาลไหนครับ
สมชาย: โรงพยาบาลศิริราชครับ
มานี: ไปตรงไหนครับ
สมชาย: ตรงข้ามวัดพระแก้วครับ
มานี: ใช่ครับ รู้แล้ว
สมชาย: ขอบคุณมากครับ
มานี: ไม่เป็นไรครับ ถึงแล้วจะบอก
สมชาย: ถึงแล้วช่วยบอกทีนะ
มานี: ได้เลยครับ อย่าลืมลงรถนะครับ
สมชาย: จะลงตรงไหนครับ
มานี: หน้าโรงพยาบาลเลยครับ
สมชาย: ขอบคุณมากครับ
```

### 2. สั่งอาหารผ่านโทรศัพท์
```
สมชาย: สวัสดีครับ ร้านอาหารสุขใจ
มานี: สวัสดีค่ะ สั่งอะไรได้ครับ
สมชาย: สั่งข้าวผัดไทย 1 จาน
มานี: มีอะไรเพิ่มเติมไหมครับ
สมชาย: ไข่ตุ๋น 1 จาน
มานี: มีกะเทียมหรือเปล่าครับ
สมชาย: ไม่เอาครับ
มานี: รวมเป็นเท่าไหร่ครับ
สมชาย: 150 บาทครับ
มานี: ส่งที่ไหนได้ครับ
สมชาย: ถนนสุขุมวิทร ซอย 5 ครับ
มานี: ใช่ครับ รอ 30 นาทีนะครับ
สมชาย: ได้ครับ ขอบคุณครับ
มานี: ขอบคุณครับ รอสินค้าได้เลยครับ
```

## 📚 บทสนทนาการศึกษา (3-4 นาที)

### 1. คุยครูเรียน
```
สมชาย: ครูครับ มีเรื่องจะปรึกษา
มานี: อะไรเหรอครับ สมชาย
สมชาย: เรื่องการบ้านครับ
มานี: ไม่เข้าใจตรงไหนเหรอครับ
สมชาย: บทที่ 3 เรื่องเศษฐศาสตร์
มานี: อ๋อ ทำไมไม่เข้าใจล่ะครับ
สมชาย: คำศัพท์ยากไปหน่อยครับ
มานี: อย่าลืมถามครูได้เสมอ
สมชาย: จะจำไว้เลยครับ
มานี: พรุ่งนี้ครูจะอธิบายใหม่
สมชาย: ขอบคุณครูครับ
มานี: เป็นห่วงนะครับ
```

### 2. ทบทวนบทเรียน
```
สมชาย: มานีครับ ทบทวนบทเรียนกัน
มานี: ได้ครับ ครูสมชาย
สมชาย: บทที่ผ่านมาเข้าใจไหม
มานี: เข้าใจบางส่วนครับ
สมชาย: ส่วนไหนที่ยังไม่เข้าใจ
มานี: ส่วนของคณิตศาสตร์ครับ
สมชาย: ลองทำโจทย์ดูไหม
มานี: ลองแล้วครับ แต่ยังไม่ได้
สมชาย: ครูจะสอนใหม่ให้
มานี: ขอบคุณมากครับ
สมชาย: พยายามอีกนะครับ
มานี: จะพยายามให้สุดความสามารถครับ
```

## 🏥 บทสนทนาทางการแพทย์ (3-4 นาที)

### 1. นัดหมอ
```
สมชาย: สวัสดีครับ คลินิกหมอใจ
มานี: สวัสดีค่ะ มีอะไรให้ช่วยไหมครับ
สมชาย: อยากนัดหมอครับ
มานี: นัดหมอวันไหนครับ
สมชาย: วันพรุ่งนี้ตอนบ่ายครับ
มานี: หมอคนไหนครับ
สมชาย: หมอสมชายครับ
มานี: มีอาการอะไรครับ
สมชาย: ปวดหัวและไข้ว่า
สมชาย: มาตั้งแต่เมื่อวานครับ
มานี: ต้องมาก่อน 30 นาทีครับ
สมชาย: ได้ครับ ขอบคุณครับ
มานี: ไม่เป็นไรครับ ดูแลตัวเองด้วยนะครับ
```

### 2. ปรึกษาอาการ
```
สมชาย: ครูมานีครับ มีอาการปวดท้อง
มานี: ปวดมานานแล้วหรือเปล่าครับ
สมชาย: 2 วันแล้วครับ
มานี: มีอาการอื่นร่วมด้วยไหมครับ
สมชาย: คลื่นไส้อยากอาเจียนครับ
มานี: ทานอาหารอะไรไปบ้างครับ
สมชาย: กินข้าวผัดมาครับ
มานี: อาจเป็นอาหารเป็นพิษครับ
สมชาย: ต้องทำอย่างไรครับ
มานี: งดอาหารทอดไปก่อนครับ
สมชาย: ได้ครับ จะทำตาม
มานี: ถ้าไม่ดีขึ้นต้องมาพบหมอนะครับ
สมชาย: ขอบคุณครับครู
```

## 🎯 วิธีใช้งาน

### 1. เลือกบทสนทนา
- เลือกบทที่เหมาะกับความยาวที่ต้องการ
- คัดลอกข้อความทั้งหมด

### 2. ตั้งค่าใน Gemini Dialogue Mode
- **Speaker 1 Name:** สมชาย (หรือชื่ออื่น)
- **Speaker 1 Voice:** เลือกเสียงผู้ชาย (เช่น Kore, Puck)
- **Speaker 2 Name:** มานี (หรือชื่ออื่น)  
- **Speaker 2 Voice:** เลือกเสียงผู้หญิง (เช่น Aoede, Lotus)

### 3. ปรับแต่ง Tone Instruction
```
พูดด้วยน้ำเสียงเป็นกันเอง
Speak in a friendly conversational tone
พูดด้วยน้ำเสียงเป็นทางการ
Speak in a professional formal tone
พูดด้วยน้ำเสียงสดใสและมีชีวิตชีวา
Speak with an energetic and lively tone
```

### 4. ทดสอบและปรับปรุง
- ฟังผลลัพธ์และปรับชื่อตัวละคร
- ลองเปลี่ยนเสียงเพื่อเปรียบเทียบ
- ปรับ tone instruction ให้เหมาะกับบทสนทนา

## 💡 เคล็ดลับเพิ่มเติม

### การเขียนบทสนทนาที่ดี:
- **ชื่อตัวละคร:** ควรชัดเจนและแตกต่างกัน
- **บทสนทนา:** เป็นธรรมชาติ ไม่ยาวเกินไป
- **อารมณ์:** สอดคล้องกับสถานการณ์
- **ความยาว:** 2-4 นาทีเหมาะสำหรับทดสอบ

### การเลือกเสียง:
- **ผู้ชาย:** Kore, Puck, Charon, Fenrir
- **ผู้หญิง:** Aoede, Lotus, Leda, Calliope
- **ความเข้ากัน:** เลือกเสียงที่มีโทนเสียงแตกต่างกัน

### การใช้ Tone Instructions:
- **ทั่วไป:** ไม่ต้องใส่ก็ได้
- **เฉพาะเจาะจง:** ใส่เพื่อความหลากหลายในอารมณ์
